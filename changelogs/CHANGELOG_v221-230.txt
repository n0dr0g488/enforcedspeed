ENFORCEDSPEED — CHANGELOG (v221–v230)

v224 — 2026-02-05

v227 — 2026-02-05
- County import: fix psql merge step by writing SQL to a temp file and running psql -f (avoids Windows/PowerShell quoting causing "invalid command \").


v228 — 2026-02-05

v229 — 2026-02-05
